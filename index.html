HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • There are both parent and child elements that are found in code.
  • The parent element is the one that hold all the elements (Child Elements) nested into one element.
  • Its one of three cornerstone languages in web development (HTML, CSS, and JS are the main).
  • HTML is made up of elements to provide content and framework to a website.
  • Elements refer to the whole element from opening to closing tag.
  • Tag refers to what appears inside the brackets of an element.
  • Semantic elements describes its content and the role it plays on a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS stands for Cascading Style sheets which is a programing language.
  • CSS defines colors, aesthetics, organization and visual layout.
  • CSS allows the coders to format in multiple style options.
  • Inline CSS is a style in which we want to directly style in the HTML file; this is applied to a specific HTML tag and is edited individually.
  • Internal CSS is found in the head portion on an HTML file; this helps style multiple elements within a page.
  • External CSS is the most common way to style; keeps all rules in a seperate style sheet file to keep the design simple.
  • Declarations contain the value and the property of an element such as color (value): blue (property).
  • You must place a link piece of code between the style sheet and the assets file in order for style to appear.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • API's are one of the many tools that are built into web browsers.
  • A console is a test enviroment that developers use to check out their code.
  • .log() is a method of a set of instructions to be executed by a computer that has a message inside of the parenthesis.
  • Control Flow can be broken through conditional statements and loops.
  • An array is a group of related data found in a variable. They can be indexed (from 0 and put in logical order) and determined by the array's length
